# CogCache

[CogCache](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/touchcastinc1655995956899.touchcast_cogcache) is the most cost-effective way to access Azure OpenAI with no capacity limits.

By leveraging a global network of Azure supercomputing clusters, CogCache provides the highest performance and lowest cost on the market for Azure OpenAI tokens, enabling companies to unlock the full potential of generative AI.

## Installation and Setup

First, you need to get the `API_KEY` from the [CogCache](https://app.cogcache.com/) website.
